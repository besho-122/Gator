## Overview
Gator is a command-line RSS feed aggregator that fetches posts from RSS feeds, stores them in a PostgreSQL database, and allows users to browse posts directly from the terminal.

---

##  Features
- User authentication  
- Follow RSS feeds  
- Background feed aggregation  
- Store posts in database  
- Browse latest posts from followed feeds  

---

## Requirements
- Node.js (v18+)  
- PostgreSQL  
- npm  
- Git
